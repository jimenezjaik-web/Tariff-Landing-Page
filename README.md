# Tariff Landing Page

This repository contains a responsive landing page designed to present pricing or tariff information in a clean and structured way.

The project focuses on modern front-end layout techniques using HTML and CSS, emphasizing clarity, usability, and visual hierarchy.

## Live Preview
👉 https://jimenezjaik-web.github.io/Tariff-Landing-Page/

## Key Features
- Clean and modern landing page design
- Responsive layout optimized for desktop and mobile
- Well-structured pricing/tariff sections for easy comparison
- Clear call-to-action elements
- Consistent typography and spacing system

## Technical Notes
- Built with semantic HTML5 and CSS3
- Uses Flexbox and/or Grid for layout structure
- No external frameworks or libraries
- Focused on clean, maintainable code

## Use Case
This project represents a typical pricing or service landing page, commonly used for SaaS products or service-based businesses. It demonstrates practical front-end development skills and the ability to build clear, user-focused interfaces.

📌 **Note:** This is a static front-end project and does not include backend functionality or dynamic interactions.
